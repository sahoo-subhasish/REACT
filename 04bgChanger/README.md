<h2>Today's Implementation — Background Color Changer</h2>

<p>
Created a simple background color switcher to understand React state updates and component communication (parent → child props).
Buttons dynamically update the page background when clicked.
</p>

<h3>Concepts Practiced</h3>

<ul>
<li><b>useState</b> — managed global background color state</li>
<li><b>Props</b> — passed state updater function to child components</li>
<li><b>Component Re-rendering</b> — UI updates automatically when state changes</li>
<li><b>Event Handling</b> — handled click events in child components</li>
<li><b>Component Reusability</b> — reused a single Button component with different values</li>
</ul>

<h3>Features</h3>

<ul>
<li>Multiple selectable colors</li>
<li>Instant background update</li>
<li>Reusable button component</li>
<li>Clean parent-child interaction</li>
</ul>
