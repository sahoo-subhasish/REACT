<h1>Currency Converter — React Project</h1>

<p>
A real-time Currency converter built using React.  
This project converts an entered amount from one currency to another using live exchange rates fetched from an external API.
</p>

<hr/>
<h2>Features</h2>

<ul>
<li>Convert between multiple world currencies</li>
<li>Live exchange rates from API</li>
<li>Swap currencies instantly</li>
<li>Reusable input component</li>
<li>Auto-updating dropdown currency list</li>
</ul>

<hr/>

<h2>Concepts Learned</h2>

<ul>
<li><b>Custom Hooks</b> — fetching API data using a reusable hook</li>
<li><b>useEffect</b> — fetching exchange rates when currency changes</li>
<li><b>useState</b> — managing amount, selected currencies, and conversion</li>
<li><b>Props & Component Reusability</b> — same input component used for both fields</li>
<li><b>Form Handling</b> — submit-based conversion</li>
<li><b>Dynamic Rendering</b> — dropdown generated from API data</li>
<li><b>useId</b> — unique input-label linking</li>
</ul>
